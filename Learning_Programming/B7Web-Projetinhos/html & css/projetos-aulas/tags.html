Tags Básicas

Temos os "níveis" de titulos da página que são:

h1, h2, h3, h4, h5, h6

Quanto maior o h menor ele vai ser

Eles saõ usados por motores de busca do google por exemplo




Temos também a tag p que é essa que está sendo utilizada para paragrafo




Também temos a tag de ancora que é: a e ela é acompanhada por href:"link aqui" para redirecionar para algum link

Ir para o Youtube...


Temos também a tag para colocar imagens na página, pode ser colocada por link da imagem ou ter uma na pasta do seu projeto

Ela também é acompanhada por alt:"" para ter uma explicação caso não tenha carregado a imagem ou para pessoas com deficiência visual

Temos uma tag para botões, que o propio nome já diz




Uma tag muito importante é a br que é para quebra uma linha (pular no caso ou ir para a proxima)




Temos também as tags de listagem que saõ ul para lista desordenadas e a li que é o item da lista

  • Item 1
  • Item 2
  • Item 3

Tambem tem a lista ordenada que ao invés de ul é ol

  1. Item 1
  2. Item 2
  3. Item 3
